## üè™ Setting Up Your Store

1. Navigate to **Dashboard** ‚Üí **Create Store**


## üí≥ Payment Configuration

### Step 3: Set Up Payment Methods

Navigate to **Dashboard** ‚Üí **Settings** ‚Üí **Payment**

You can accept payments through 4 different methods:

#### Option 1: Stripe (Recommended) üí≥
**Best for:** Credit cards, Apple Pay, Google Pay
**Fee:** 2.9% + $0.30 per transaction
**Setup Time:** 10 minutes

**How to Set Up:**
1. Select **"STRIPE"** as primary processor
2. Click **"Connect Stripe"**
3. Complete Stripe Connect onboarding
4. Verify your bank account
5. Done!

#### Option 2: PayPal üÖøÔ∏è
**Best for:** PayPal customers
**Fee:** 2.9% + $0.30 per transaction
**Setup Time:** 5 minutes

**How to Set Up:**
1. Select **"PAYPAL"** as processor
2. Enter your PayPal email address
3. (Optional) Enter PayPal Merchant ID
4. Save settings

#### Option 3: Square ‚¨õ
**Best for:** Square POS integration
**Fee:** 2.6% + $0.10 per transaction
**Setup Time:** 10 minutes

**How to Set Up:**
1. Select **"SQUARE"** as processor
2. Get Square Access Token from [developer.squareup.com](https://developer.squareup.com)
3. Get your Location ID from Square dashboard
4. Enter credentials and save

#### Option 4: Cash (Pickup) üíµ
**Best for:** Local pickup sales
**Fee:** $0 processing fee
**Setup Time:** 2 minutes

**How to Set Up:**
1. Select **"CASH"** as processor
2. Check **"I accept cash payments"**
3. Enter pickup instructions:
   ```
   Example:
   "Pickup available at 123 Main St, Chicago IL
   Monday-Friday 9am-5pm
   Please call (555) 123-4567 when you arrive
   Bring exact change"
   ```
4. Save settings

### Platform Fees
All payment methods include a **7% platform fee** (deducted from your payout).

**Example:** $100 sale
- Stripe fee: $3.20
- Platform fee: $6.78 (7% of $96.80)
- **Your payout: $90.02**

## üì¶ Adding Products

**Basic Details:**
- **Product Name** (e.g., "Premium Stepping Shoes - Black")

- Custom variants

## üì¨ Managing Orders

### Step 5: Process Your Orders

#### When You Receive an Order

1. **You'll receive an email notification**
   - Order number
   - Customer information
   - Items ordered
   - Shipping address

2. **Check your dashboard**
   - Go to **Dashboard** ‚Üí **Orders**
   - View order details
   - Print packing slip

3. **Prepare the order**
   - Package items securely
   - Include packing slip
   - Add any promotional materials

4. **Ship the order**
   - Use your preferred shipping carrier
   - Get tracking number
   - Update order in dashboard

#### Mark Order as Fulfilled

1. Go to **Dashboard** ‚Üí **Orders** ‚Üí Select order
2. Click **"Mark as Fulfilled"**
3. Enter:
   - Shipping carrier (USPS, UPS, FedEx, etc.)
   - Tracking number
4. Click **"Save"**

**Customer automatically receives:**
- Shipping confirmation email
- Tracking number
- Estimated delivery date

### Order Timeline

```
1. Order Placed ‚Üí You receive email alert
2. Payment Processed ‚Üí Funds held by platform
3. You Ship Order ‚Üí Update with tracking
4. Customer Receives ‚Üí Review request sent (7 days later)
5. Payout Released ‚Üí Next payout cycle
```

### Payout Schedule

- **Frequency:** Weekly (every Friday)
- **Method:** Bank transfer (ACH)
- **Processing Time:** 2-5 business days
- **Minimum:** $25

## ‚ùì Frequently Asked Questions

### General Questions

**Q: How much does it cost to become a vendor?**
A: Free! No setup fees or monthly fees. You only pay when you make a sale (7% platform fee + payment processor fees).

**Q: When do I get paid?**
A: Payouts are processed weekly (every Friday) for orders delivered in the previous week. Funds typically arrive in your bank account within 2-5 business days.

**Q: What products can I sell?**
A: Any legal products related to stepping, dance, fashion, or lifestyle. Prohibited items include counterfeit goods, weapons, illegal items, and adult content.

**Q: Can I sell outside the United States?**
A: Currently, we only support shipping within the United States.

### Order Management

**Q: What if a customer wants to return an item?**
A: Handle returns according to your store policy. You can offer refunds, exchanges, or store credit. Update the order status in your dashboard accordingly.

**Q: What if an item is out of stock?**
A: Contact the customer immediately, offer alternatives or a refund. Update your inventory in the dashboard to prevent future orders.

**Q: Can I cancel an order?**
A: Orders can be cancelled before shipping. Contact support@stepperslife.com to process the cancellation and refund.

### Payment Questions

**Q: Why can't I use multiple payment processors?**
A: You can! Set a primary processor (required) and add a secondary processor (optional) as a backup payment method for customers.

**Q: How do refunds work?**
A: Refunds are processed through the same payment method used for the original purchase. Platform fees are not refunded.

**Q: What if my payout is delayed?**
A: Check your Stripe/PayPal dashboard for any issues. Contact support if payouts are delayed more than 7 business days.

### Technical Questions

**Q: Can I integrate my existing inventory system?**
A: API integration is coming soon. For now, manage inventory through the dashboard.

**Q: Can I export my sales data?**
A: Yes! Go to Dashboard ‚Üí Analytics ‚Üí Export Data to download CSV reports.

**Q: Is there a mobile app?**
A: The dashboard is mobile-responsive. Native mobile apps are planned for future release.
